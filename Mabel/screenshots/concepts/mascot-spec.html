Mabel — Asset Specifications

Mascot, Icon & Wordmark export guide for the design uplift

Generated April 16, 2026 — Reference document for asset production

1. MabelMascot — Full Body Character

The full-body mascot is the hero visual element. It appears at multiple sizes across the app, always loaded from a single MabelMascot.imageset asset and scaled down via .resizable().scaledToFit().

Where It's Used

ScreenDisplay SizeContext
Welcome (hero)260ptCentered hero, largest instance
Library (greeting header)140ptLeft-aligned next to greeting text
My Stories (empty state)80ptCentered, with helper text below

Export Specifications

ScaleMinimum WidthFilename
@1x260pxmabel-mascot.png
@2x520pxmabel-mascot@2x.png
@3x780pxmabel-mascot@3x.png
Format: PNG with transparent background.
Aspect ratio: Preserve the character's natural proportions — SwiftUI uses .scaledToFit() so it will letterbox within the frame.
Rendering: .renderingMode(.original) — original colors are preserved, no tinting.
Recommendation: Design at @3x (780px wide minimum) and export downscaled versions for @2x and @1x.

Destination Path

Mabel/Mabel/Assets.xcassets/MabelMascot.imageset/

2. MabelIcon — Circular Head Crop

A cropped head/face version of the mascot, displayed inside the MabelWordmarkLockup component (icon + wordmark text) on screens 2–8. The image is clipped to a circle in code.

Where It's Used

ComponentDisplay SizeContext
MabelWordmarkLockup28pt circleTop of Setup, RecordingSetup, Recording, ChapterReview, MyStories, Profile

Export Specifications

ScaleSize (px)Filename
@1x28 × 28MabelIcon.png
@2x56 × 56MabelIcon@2x.png
@3x84 × 84MabelIcon@3x.png
Format: PNG with transparent background.
Shape: Export as a square — the circle clip is applied in code via .clipShape(Circle()).
Content: Head/face crop of the mascot, centered within the square. Leave a small margin so the face doesn't touch the circle edge.
Minimum source: 84px (the @3x size) — but 252px+ recommended for future-proofing.

Destination Path

Mabel/Mabel/Assets.xcassets/MabelIcon.imageset/

3. AppIcon — Home Screen & App Store

The iOS app icon displayed on the home screen and in the App Store.

Export Specifications

SizeFormatNotes
1024 × 1024pxPNGNo transparency, no baked rounded corners
No transparency: iOS requires opaque app icons — use a solid background color.
No rounded corners: iOS applies the superellipse mask automatically.
Background: Solid brand color or white — whichever showcases the mascot best.
Content: Mascot head or simplified mark that reads well at 60×60pt (smallest home screen render).

Destination Path

Mabel/Mabel/Assets.xcassets/AppIcon.appiconset/

4. MabelWordmark — Logo Text

The "Mabel" text logo, used at the top of every screen. This is an image asset, not rendered as a Text() view.

Where It's Used

ComponentDisplay HeightContext
MabelWordmark24ptStandalone on Welcome and Library top bars
MabelWordmarkLockup24pt (alongside 28pt icon)Screens 2–8 top bars

Export Specifications

ScaleHeight (px)Filename
@1x24px tallmabel_type.png
@2x48px tallmabel_type@2x.png
@3x72px tallmabel_type@3x.png
Format: PNG with transparent background.
Rendering: .renderingMode(.original) — preserves original colors.
Width: Flexible — determined by the text proportions. Height is the fixed dimension.
Current state: Only @1x exists today. If redesigning, export all 3 scales.

Destination Path

Mabel/Mabel/Assets.xcassets/MabelWordmark.imageset/

Banned colors: No orange (#ed7d3c), no old peach (#F3E0D2), no old teal (#1F7A6F), no pure black (#000000).

6. Export Checklist

When the final mascot design is ready, export these files:

MabelMascot (Full Body)

  • mabel-mascot.png — @1x, 260px wide minimum, transparent PNG
  • mabel-mascot@2x.png — @2x, 520px wide, transparent PNG
  • mabel-mascot@3x.png — @3x, 780px wide, transparent PNG

MabelIcon (Head Crop)

  • MabelIcon.png — @1x, 28×28px, transparent PNG
  • MabelIcon@2x.png — @2x, 56×56px, transparent PNG
  • MabelIcon@3x.png — @3x, 84×84px, transparent PNG

AppIcon

  • AppIcon.png — 1024×1024px, opaque PNG, no rounded corners

MabelWordmark (if redesigning)

  • mabel_type.png — @1x, 24px tall, transparent PNG
  • mabel_type@2x.png — @2x, 48px tall, transparent PNG
  • mabel_type@3x.png — @3x, 72px tall, transparent PNG
Total files: 7 minimum (3 mascot + 3 icon + 1 app icon), up to 13 if wordmark and icon also change.
Delivery: Drop files into the corresponding .imageset directories and the asset swap takes ~30 seconds.
